# Layout

## File manifests

```bash
Repository
│ 
└───apps
      manifest_1.yaml
      manifest_2.yaml
      ...
```

## Directory manifests

```bash
Repository
│ 
└───apps
    │   manifest_1.yaml
    │   manifest_2.yaml
    │   ...
    │
    └───manifest_3
        │ main.yaml
        │
        └───files
              config       
```
